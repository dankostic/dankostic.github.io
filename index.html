PORTRET

 Sportret is a social network project. This project was built from scratch, with using of Codeigniter framework, Bootstrap, JS, AJAX, jQuery.

 It has special accounts designed for users, as well as for clubs, athletes, federations, media, municipalities etc... which are accounts for pages. Sportret also has its own admin panel, and CMS for other users.

Currently this project is on hibernated state, but there are available snapshots on web.archive.org
